Expected Format

One row per drillhole interval. Required columns: Hole ID, X, Y, Z (desurveyed coordinates), FROM, TO. Values below detection limit (e.g. <0.5) are automatically converted to half the detection limit.

⚠ MISSING GRADE VALUES DETECTED

Some intervals have missing or null assay values. Choose how to handle them:

Leave Blank
Missing values are excluded from calculations — intervals contribute length only (recommended)
Treat as Zero
Missing values become 0 — use only if absence of detection genuinely means zero grade
Assign Background Grade
Missing values are replaced with the dataset median for that element — preserves grade continuity
